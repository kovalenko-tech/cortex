What Cortex does

Deep project intelligence

Four analysis layers that give your coding agent context no human could maintain manually.

🔍

Git history mining

Tracks every bug fix, hotfix, and refactor in your codebase. Claude knows which files have a troubled past before touching them.

🔗

Co-change analysis

Statistically identifies which files break together. Change auth.py? Cortex knows you also need to update middleware/session.py.

🔒

Security audit

SAST analysis per file with secret scanning across your entire git history. Issues are surfaced in context — not buried in a separate report.

🤖

AI summaries

Optional Claude Haiku-powered overview per file. Explains purpose, patterns, and gotchas in plain language — accelerating cold starts.

📦

Dependency Scan

Checks Python, Node.js and Flutter dependencies for known vulnerabilities. Results go straight into .claude/docs/.

🔌

MCP Server

Native Claude Code integration via Model Context Protocol. Context is injected automatically — no manual file reading.

⚡

Smart Caching

Unchanged files are skipped on re-runs using MD5 file hashing. Only modified files are re-analyzed — repeat runs are near-instant.

🔄

Parallel Analysis

Analyzes up to 8 files simultaneously using thread pool execution. Large projects run 5-8x faster than sequential analysis.

📊

Risk Scoring

Every file gets a risk score (0–100) based on bug history, test coverage, security issues and how many files depend on it. Claude Code prioritizes carefully.

💬

PR Knowledge

Mines GitHub PR comments and discussions for implicit decisions: why Redis was chosen, what breaks, what's intentional. Knowledge that lives nowhere in code.

Native integration

Native Claude Code Integration

Add cortex as an MCP server — Claude Code gets context automatically for every file you open.

.claude/settings.json
{
  "mcpServers": {
    "cortex": {
      "command": "cortex",
      "args": ["mcp"]
    }
  }
}

CLI reference

All commands

Every tool you need, nothing you don't.

Command Description
cortex setup
interactive setup wizard
cortex analyze
analyze full project
cortex analyze --since HEAD~20
only changed files
cortex analyze --no-llm
skip AI summaries (faster)
cortex analyze --no-cache
force full re-analysis
cortex analyze --max-files 100
limit files analyzed
cortex init
generate CLAUDE.md for your project
cortex context <file>
show context for one file
cortex security
security audit only
cortex deps
scan dependencies for vulnerabilities
cortex risks
show risk scores for all files
cortex freshness
show how stale the context is
cortex mine-prs
mine GitHub PR decisions
cortex diff main
update context for changed files
cortex status
show analysis status
cortex clean
remove all generated context files
cortex install-hook
auto-update on every commit
cortex watch
auto-update when commits arrive
cortex mcp
start MCP server for Claude Code

Flags

--no-llm
skip AI summaries (no API key needed)
--no-cache
force full re-analysis
--max-files N
limit files analyzed
--since REF
only changed files since git ref
