Documentation

Complete reference for Cortex CLI — all commands, flags, and configuration.

Installation

curl -fsSL https://raw.githubusercontent.com/kovalenko-tech/cortex/main/install.sh | bash

Or with npx (requires Python 3.11+):

npx cortex-ai analyze

Commands

Setup & Analysis

CommandDescription
cortex setupInteractive wizard — analyze, configure MCP, install git hook
cortex analyzeAnalyze full project and generate .claude/docs/
cortex initGenerate CLAUDE.md with project overview and commands
cortex mine-prsMine GitHub PR comments for implicit architectural decisions

Query & Inspect

CommandDescription

cortex statusShow analysis status, freshness summary, risk overview
cortex risksList files by risk score (HIGH / MEDIUM / LOW)

cortex securityRun security audit across all files

Maintenance

CommandDescription

Flags

FlagCommandDescription
--since <ref>analyzeOnly analyze files changed since git ref (e.g. HEAD~10, main)
--no-llmanalyzeSkip AI summaries — faster, no API key needed
--no-cacheanalyzeForce full re-analysis ignoring file cache
--max-files NanalyzeLimit number of files analyzed
--langanalyzeFilter by language: python, js, ts, dart, go
--forceinitOverwrite existing CLAUDE.md
--tokenmine-prsGitHub token (or set GITHUB_TOKEN env var)
--max-prs Nmine-prsMax number of PRs to analyze (default: 50)
--levelrisksFilter by risk level: HIGH, MEDIUM, LOW, all
--stale-onlyfreshnessShow only stale and outdated files

MCP Integration

Add to .claude/settings.json in your project:

{
  "mcpServers": {
    "cortex": {
      "command": "cortex",
      "args": ["mcp"]
    }
  }
}

Restart Claude Code. Context is now injected automatically when you open any file.

GitHub PR Knowledge

export GITHUB_TOKEN=ghp_...
cortex mine-prs
cortex analyze

Cortex mines merged PRs for inline review comments and discussion, extracting architectural decisions, warnings, and reasoning. Results appear in the ## Decisions & Context section of each file's context.

AI Summaries

export ANTHROPIC_API_KEY=sk-ant-...
cortex analyze

Uses Claude Haiku to generate a 2–3 sentence technical overview per file. Optional — all other features work without an API key.

GitHub Action

Auto-update context on every push. See cortex.yml example.
